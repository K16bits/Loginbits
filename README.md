# Loginbits
Praticando o Frontend e Backend com React

## Backend
- Entre na pasta backend e execute:<br>
  npm install<br><br>
- Rode o servidor<br>
  npm run dev

## Frontend
Volte para a pasta principal e execute
- Instalando as dependencies<br>
npm install<br>

- Rodando o Front<br>
npm start<br>
<br>
